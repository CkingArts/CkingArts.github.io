You Are A ROBLOX Pirate

You are a ROBLOX Pirate is a MACHINIMA Animation using ROBLOX's game engine, ROBLOX Studio. The program already has an animation feature, instead I use a plugin called Moon Animator to help make the process easier. The fresh animations are spliced together and edited in Adobe Premiere Pro.

The video is a parody of a music video to the song You Are a Pirate from the show Lazy Town. This project was a way to bring comedy into a fun kids game using an early 2000s reference.
